# companion-ios
LLM based journalling app

## Features
- **On-Device AI**: Local LLM inference using MLX for privacy-focused conversations
- **Cloud AI**: Optional cloud-based models (OpenAI-compatible) for enhanced capabilities
- **Flexible Model Selection**: Switch between on-device and cloud models seamlessly
- **Streaming Responses**: Real-time token generation for both on-device and cloud modes
- **Persistent Chat History**: SwiftData-based storage with date-grouped organization

## Cloud Agent Support
Companion now supports cloud-based AI agents! See [CLOUD_AGENT.md](CLOUD_AGENT.md) for detailed documentation on:
